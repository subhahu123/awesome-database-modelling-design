# Interactive playground

Use this mini demo to **play with database design features** and see their effect instantly.

- Toggle **composite indexes** to observe read latency changes.
- Toggle **idempotency keys** to see duplicate-write risk under retries.
- Toggle **audit/history trail** to understand compliance readiness trade-offs.
- Increase **QPS** and retries to simulate scale pressure.
